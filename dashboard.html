Tentang CoffeeVision AI

CoffeeVision AI merupakan aplikasi berbasis Deep Learning yang digunakan untuk mengklasifikasikan tingkat roasting biji kopi menggunakan model MobileNetV2.

Pengguna hanya perlu mengunggah gambar, kemudian sistem akan melakukan analisis secara otomatis dan menampilkan hasil prediksi beserta tingkat keyakinan model AI.

🤖 Teknologi

Flask • TensorFlow • MobileNetV2

🧠 Metode

Transfer Learning

📷 Dataset

Coffee Bean 224 Dataset

💻 Bahasa

Python • HTML • CSS • JavaScript
